Como funcionam os comandos e acessos

No geral, os comandos servem para acessar o painel de edição, painel de membros e o chat

Líder: pode editar, gerenciar membros, adicionar ou remover e alterar senhas.

Membro: pode editar e visualizar os outros membros.

Ambos podem usar o chat de feedbacks e ideias para sugerir melhorias ou relatar bugs do site, vídeo mostrando os comandos pra entrar no modo de edição e pro chat.

Funções do painel de edição

O painel de edição é uma das partes mais importantes do site dos projetos. Ele permite modificar informações tanto visual quanto das infos do projeto. Abaixo vou mostrar como cada função funciona e pra que servem

Como funciona a edição das informações

Cada projeto vem como padrão (Grupo: , Classe: , Resumo: , Intenção: e Finalidade:) mas pode ser alterado tanto pra adicionar novas infos quanto pra remover as que achar desnecessárias, podendo ter um maior controle do que quer no seu Projeto sem se limitar a um padrão

Como funciona a adição de Mídia

Você pode inserir imagens, vídeos e outros formatos para deixar o projeto mais completo. O limite é de até 5 tipos de mídia por projeto, garantindo organização e bom desempenho.

Como funciona a Alteração das cores de containers e textos de cada Projeto

Cada projeto pode personalizar a cor de vários containers do site que se adaptam ao Projeto tanto quanto alteração da cor do texto das infos.

Adicionar som de fundo

Cada Projeto poderá adicionar um som de fundo que se adapta a cada Projeto

Painel de líderes e membros

O painel de líderes e membros é responsável pela organização do grupo.

Líderes podem gerenciar senhas, adicionar ou remover membros, e acessar todas as edições.

Membros podem editar seus projetos e visualizar quem participa do grupo.

Chat de feedbacks e ideias

O chat será para compartilhar ideias e relatar problemas no site. Também conectando todos os membros e líderes, facilitando a comunicação e as melhorias no site.

Use sempre o Chat para reportar erros e dar ideias de melhorias pro site.
